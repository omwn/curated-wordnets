# curated-wordnets
A curated list of wordnets for people who want many

My goal is to find and make accesible as many wordnets as possible.

Ideally for each wordnet it should be:
- in GWA LMF format (validated)
- with no errors (according to wn)
- with no loops (according to OMW and cygnet)
- released under an open license
- of high quality (hard to check automatically)

For each wordnet we should have some basic info

- url for wordnet
   - ideally as a package - with LICENSE, README and citation
   see https://wn.readthedocs.io/en/latest/guides/lexicons.html#wn-lmf-files-packages-and-collections
- validation status
- type: expand/merge/extend/auto
- size information
- rating: In/Out (our subjective judgement)
- does it supersede an earlier wordnet

We should idealy feed information into
- GWA Wordnets in the World
- wn module index.toml
- cygnet wordnets.toml

## How to find them?
- look for wordnets in github
- look for wordnets in Clarin
- look in the GWA wordnets in the world list

Once we have them
- if it is in OMW 1.0 format, clone to omwn, convert and PR

- validate, provide feedback, rate
